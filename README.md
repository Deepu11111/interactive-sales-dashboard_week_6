# interactive-sales-dashboard_week_6
Interactive sales analytics dashboard built using Python, Pandas, and Plotly to visualize KPIs, trends, and regional performance.
# Features
- Monthly Sales Trend (Line Chart)
- Sales by Region (Bar Chart)
- Product Distribution (Pie Chart)
- Quantity vs Sales (Scatter Plot)
- KPI Metrics (Total Sales, Quantity, Customers)

# Technologies Used
- Python
- Pandas
- Plotly

# How to Run
1. Install dependencies:
   pip install pandas plotly

2. Run:
   python dashboard.py

3. Open:
   final_sales_dashboard.html
